## Correr proyecto

Crear un virtual envoriment. Se recomienda usar la version de 3.11.\* de python

```bash
python -m venv venv
```

Activar entorno virtual .\env\Scripts\activate

```bash
.\venv\Scripts\active
```

```bash
pip install -r requirements.txt
```
